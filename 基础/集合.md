# 集合

Java集合框架是Java编程中非常重要的一部分，它提供了一套接口和类来存储和操作数据。

## 理解集合

怎么理解集合呢，我感觉他是一种扩展了数组功能的数据结构。集合的出现解决了数组在开发中的诸多局限性：数组一旦定义长度不可变，但是集合可以自动扩容，数组只能存储相同类型的数据，集合可以通过泛型实现不同数据类型的存储……

**动态扩容**：数组长度固定，而集合可以动态调整大小

**丰富操作**：集合提供了丰富的API（添加、删除、查找、排序等）

**数据类型**：数组只能存储相同类型数据，集合可以存储不同类型（通过泛型控制）

**数据结构多样性**：提供多种数据结构实现（链表、哈希表、树等）

## List接口及其实现类

### ArrayList示例

```java
import java.util.ArrayList;
import java.util.List;

public class ArrayListDemo {
    public static void main(String[] args) {
        // 创建ArrayList
        List<String> fruits = new ArrayList<>();
        
        // 添加元素
        fruits.add("Apple");
        fruits.add("Banana");
        fruits.add("Orange");
        
        // 访问元素
        System.out.println("第一个水果: " + fruits.get(0));
        
        // 遍历List
        System.out.println("所有水果:");
        for (String fruit : fruits) {
            System.out.println(fruit);
        }
        
        // 删除元素
        fruits.remove("Banana");
        System.out.println("删除Banana后: " + fruits);
        
        // 检查大小
        System.out.println("列表大小: " + fruits.size());
    }
}
```

### LinkedList示例

```java
import java.util.LinkedList;
import java.util.List;

public class LinkedListDemo {
    public static void main(String[] args) {
        // 创建LinkedList
        List<Integer> numbers = new LinkedList<>();
        
        // 添加元素
        numbers.add(10);
        numbers.add(20);
        numbers.add(30);
        
        // 在开头添加元素
        ((LinkedList<Integer>) numbers).addFirst(5);
        
        // 在末尾添加元素
        ((LinkedList<Integer>) numbers).addLast(40);
        
        System.out.println("链表内容: " + numbers);
        
        // 获取第一个和最后一个元素
        System.out.println("第一个元素: " + ((LinkedList<Integer>) numbers).getFirst());
        System.out.println("最后一个元素: " + ((LinkedList<Integer>) numbers).getLast());
    }
}
```

## Set接口及其实现类

### HashSet示例

```java
import java.util.HashSet;
import java.util.Set;

public class HashSetDemo {
    public static void main(String[] args) {
        // 创建HashSet
        Set<String> cities = new HashSet<>();
        
        // 添加元素
        cities.add("Beijing");
        cities.add("Shanghai");
        cities.add("Guangzhou");
        cities.add("Beijing"); // 重复元素不会被添加
        
        System.out.println("城市集合: " + cities);
        
        // 检查元素是否存在
        System.out.println("包含Shanghai吗? " + cities.contains("Shanghai"));
        
        // 删除元素
        cities.remove("Guangzhou");
        System.out.println("删除Guangzhou后: " + cities);
    }
}
```

### TreeSet示例

```java
import java.util.Set;
import java.util.TreeSet;

public class TreeSetDemo {
    public static void main(String[] args) {
        // 创建TreeSet
        Set<Integer> numbers = new TreeSet<>();
        
        // 添加元素(自动排序)
        numbers.add(50);
        numbers.add(20);
        numbers.add(80);
        numbers.add(10);
        
        System.out.println("排序后的数字: " + numbers);
        
        // 获取第一个和最后一个元素
        System.out.println("最小的数字: " + ((TreeSet<Integer>) numbers).first());
        System.out.println("最大的数字: " + ((TreeSet<Integer>) numbers).last());
    }
}
```

## Map接口及其实现类

### HashMap示例

```java
import java.util.HashMap;
import java.util.Map;

public class HashMapDemo {
    public static void main(String[] args) {
        // 创建HashMap
        Map<String, Integer> studentScores = new HashMap<>();
        
        // 添加键值对
        studentScores.put("Alice", 90);
        studentScores.put("Bob", 85);
        studentScores.put("Charlie", 95);
        
        // 获取值
        System.out.println("Alice的分数: " + studentScores.get("Alice"));
        
        // 遍历Map
        System.out.println("所有学生分数:");
        for (Map.Entry<String, Integer> entry : studentScores.entrySet()) {
            System.out.println(entry.getKey() + ": " + entry.getValue());
        }
        
        // 检查键是否存在
        System.out.println("包含David吗? " + studentScores.containsKey("David"));
        
        // 删除键值对
        studentScores.remove("Bob");
        System.out.println("删除Bob后: " + studentScores);
    }
}
```
